# 🚀 התחלה מהירה - אתר מעקב דף היומי

## הרצה מקומית

### 1. התקנת תלויות
```bash
npm install
```

### 2. הרצת האפליקציה
```bash
npm run dev
```

האפליקציה תרוץ על: `http://localhost:5173`

### 3. בנייה לייצור
```bash
npm run build
```

### 4. תצוגה מקדימה של הבנייה
```bash
npm run preview
```

## 📱 איך להשתמש

### הוספת דף ראשון
1. פתחו את האפליקציה בדפדפן
2. לחצו על "תיעוד דף חדש"
3. בחרו מסכת (למשל: ברכות)
4. הזינו מספר דף (למשל: 2)
5. בחרו תאריך ורמת הבנה
6. לחצו "שמירת לימוד"

### צפייה בסטטיסטיקות
- לחצו על "סטטיסטיקות" בתפריט העליון
- תוכלו לראות את ההתקדמות שלכם

### ניהול דפים
- לחצו על "הדפים שלי" לצפייה בכל הדפים שלמדתם
- השתמשו בחיפוש למציאת דפים ספציפיים

## 🎯 תכונות עיקריות

✅ תיעוד דפים עם פרטים מלאים  
✅ מעקב רמת הבנה (1-5)  
✅ סטטיסטיקות מתקדמות  
✅ ממשק בעברית (RTL)  
✅ שמירה מקומית (LocalStorage)  
✅ עיצוב רספונסיבי  

## 🔧 פתרון בעיות

### האפליקציה לא נטענת?
- ודאו ש-Node.js מותקן (גרסה 16+)
- הריצו `npm install` שוב
- בדקו שהפורט 5173 פנוי

### נתונים נעלמו?
- הנתונים נשמרים ב-LocalStorage של הדפדפן
- אל תנקו את ה-cache של הדפדפן
- בעתיד נוסיף גיבוי לענן
